How to rip Flash music (FSF)
16Flash music could sometimes play audio per frames, so we made a new format, named FSF (Flash Sound File).
1. Start a Flash (version 4) project with a 1x1 resolution and a black background.
2. Put music files, extracted with a Flash decompiler. Make sure to check "Use imported MP3 quality" in the properties menu, or else the quality would be low.
3. Export them as an SWF file, then change the file format to FSF.
4. Even before we made this format, Winamp supports it even by default as it thinks it was an SWF. Try tagging the songs as you can!
5. Make an info file, with the following format:
[Title]
Dumped by: [Dumper]
Tagged by: [Tagger]
Downloaded from: http://pcmusic.epizy.com/
6. Compress it into a 7z file, then change the file format to FSA (Flash Sound Archive).
